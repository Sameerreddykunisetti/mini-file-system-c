# Mini File System in C

A simple file system simulator built using C programming.

## Features

- Create files
- Write data
- Read files
- Delete files
- List files
- Persistent storage

## Concepts Used

- Structures
- File Handling
- Binary Storage
- Memory Management
- OS File System Concepts


## How To Run

Compile:

gcc src/main.c src/filesystem.c -o filesystem


Run:

./filesystem
